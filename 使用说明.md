# PaddleOCR 图片转写使用说明

## 运行时间参考

| 场景 | 耗时 |
|------|------|
| 首次运行（下载模型） | 3~5 分钟（需联网，模型约几百MB） |
| 后续运行（模型已缓存） | 30~60 秒/张图片 |

模型文件缓存在 `~/.paddlex/official_models/`，下载一次后即可离线使用。

---

## 环境管理

项目使用虚拟环境隔离依赖，位于当前目录的 `venv/` 文件夹。

```bash
# 激活虚拟环境（每次新开终端都需要执行）
source venv/bin/activate

# 退出虚拟环境
deactivate

# 查看已安装的包
pip list | grep paddle
```

---

## 启动 OCR 转写

### 单张图片转写

```bash
source venv/bin/activate
PADDLE_PDX_DISABLE_MODEL_SOURCE_CHECK=True python ocr_to_md.py 测试1.png
```

### 批量转写（多张图片）

```bash
source venv/bin/activate
PADDLE_PDX_DISABLE_MODEL_SOURCE_CHECK=True python ocr_to_md.py 图片1.png 图片2.png 图片3.png
```

> `PADDLE_PDX_DISABLE_MODEL_SOURCE_CHECK=True` 用于跳过模型源连通性检查，加快启动速度。

## 输出 Markdown 文件

运行以下脚本（`ocr_to_md.py`），会自动将识别结果保存为同名 `.md` 文件：

```python
# ocr_to_md.py
import sys
import os
from paddleocr import PaddleOCR

def ocr_to_markdown(image_path):
    ocr = PaddleOCR(
        use_doc_orientation_classify=False,
        use_doc_unwarping=False,
        use_textline_orientation=False,
        lang='ch'
    )
    result = ocr.ocr(image_path)

    lines = []
    for page in result:
        if isinstance(page, dict) and 'rec_texts' in page:
            for text in page['rec_texts']:
                if text.strip():
                    lines.append(text.strip())

    md_content = '\n\n'.join(lines)

    # 输出为同名 .md 文件
    output_path = os.path.splitext(image_path)[0] + '.md'
    with open(output_path, 'w', encoding='utf-8') as f:
        f.write(md_content)

    print(f"已保存：{output_path}")
    return output_path

if __name__ == '__main__':
    if len(sys.argv) < 2:
        print("用法：python ocr_to_md.py <图片路径> [图片路径2 ...]")
        sys.exit(1)

    for image_path in sys.argv[1:]:
        ocr_to_markdown(image_path)
```

### 示例输出

运行 `python ocr_to_md.py 测试1.png` 后，会在同目录生成 `测试1.md`，内容为识别出的文字，每段以空行分隔。

## 清洗富 Markdown（HTML 表格转普通 Markdown）

PaddleOCR 识别结果有时会输出带 inline style 的 HTML table，无法被云文档解析。
使用 `clean_md.py` 通过 pandoc 将其转换为标准 pipe table：

```bash
source venv/bin/activate

# 转换单个文件（覆盖原文件）
python clean_md.py 报告.md

# 转换并另存
python clean_md.py 报告.md 报告_clean.md

# 批量转换
python clean_md.py *.md
```

> 依赖：`pandoc`（已通过 `brew install pandoc` 安装）

---

## 注意事项

- 图片中的**水印文字**也会被识别，需手动清除
- 支持格式：PNG、JPG、JPEG、BMP、TIFF
- 中英文混合内容均可识别，`lang='ch'` 同时支持中英文
